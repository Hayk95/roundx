# News Images Directory

This directory contains images for news articles. Each article in `articles.json` references an image in this folder.

## Image Requirements

- **Format**: JPG or PNG
- **Recommended Size**: 1200x630px (for optimal display)
- **File Naming**: Match the slug from articles.json (e.g., `vehicle-shipping-costs-2025.jpg`)

## Image Sources

You can use free stock photo services like:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Pixabay (https://pixabay.com)

Search for terms like:
- "car transport truck"
- "vehicle shipping"
- "auto carrier"
- "car transportation"
- "vehicle logistics"
